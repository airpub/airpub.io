## Airpub.io

[Airpub.io](http://airpub.io) 为 Airpub 提供快速便捷的 CDN 服务，当然，这也是第一个 Airpub 应用啦 :)

### 使用 Airpub.io CDN
Airpub.io CDN 基于 GitHub Pages，只 `.css` 与 `.js` CDN，由于 GitHub Pages 不支持 CORS，因此不提供主题文件 CDN。

### 使用 多说CDN
多说CDN 提供 `.css` 和 `.js` CDN，也提供主题文件 CDN.
